No longer is the desktop PC the primary way in which most people access the web. Mobile devices are the reigning champion. This is why it is becoming increasingly important for any modern website to be built from the ground up for mobile devices first. Luckily thanks to my work in progress BSc Hons Web Degree I know exactly how to create a fully responsive, modern, accessible website.

RESPONSIVE

Now I am currently in my 3rd and final year at Edge Hill University working towards my degree and absolutely loving it. In fact, my dissertation topic covers web accessibility; I am creating a website and framework that allows a website to meet all the current, nessessary, modern Web Accessibility guidelines set by W3C and WCAG. This means including aria tags so that screen readers have full compatability and also having a clear, easy to read font. As well as high contrast colours that do not conflict with the colour of the text. Features like this could be toggled on or off at the click of a button using Javascript,CSS and HTML.
