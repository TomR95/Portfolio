Cleversteam large task

The task for this was to create a custom theme that would be more appealing to use. The exact changes I made are as follows and the reasons why:

1. The colours have been changed dramatically in order to follow the current colour scheme the Cleversteam website follows. This was so customers coming from the Cleversteam site do not actually feel as though they have left the Cleversteam ecosystem. Thus feeling more comfortable using the system.

2. The font has been changed from the default 'Arial' to 'Ubuntu' so that it stands out and is easy to read. It is also the font that the Cleversteam website uses.

3. Finally, I have added Icon glyphs to the 'Show' 'Edit' and 'Destroy' links in order to better differentiate between each option. Now with the different coloured Icon glyphs, you can easily identify at a glance what each option does without having to read the word.
